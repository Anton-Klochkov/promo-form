диалоговое окно конфиг
принимаемые параметры
  bgColor- строка --------------- в RGB | RGBA | HEX; по дефолту #ffffff
  delay- число ------------------ в миллисекундах по дефолту 2000
  promoCode- строка ------------- по дефолту promoCode
  linkPartner- строка ----------- по дефолту ссылка на ютуб
  linkBanner- строка ------------ по дефолту ссылка на изображение ютуб
  titleLink- строка ------------- строка отображающаяся вместо ссылки по дефолту ссылка на парт

все цвета могут быть переданы в формате rgb, rgba, hex
_____________________________________________________________

при добавлении в другой проект сборки создаем блок с id="сборки"
примеры:  
      для формы
  <div id="form"></div>
  <script src="main._.js"></script>

      для диалогового окна
  <div id="dialog"></div>
  <script src="main._.js"></script>

npm run 
        build:prod:dialog: делает сборку продакшен версии Далогового окна 
        build:dev:dialog: делает сборку дев версии Далогового окна
        start:dev:dialog: запускает  дев версию Далогового окна

        build:prod:form:  делает сборку продакшен версии Формы
        build:dev:form: делает сборку дев версии Формы
        start:dev:form: запускает  дев версию Формы
